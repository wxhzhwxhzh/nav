Git 命令指南

快速掌握Git版本控制核心命令

基础命令

初始化

git init

初始化Git仓库

克隆

git clone <url>

克隆远程仓库

状态

git status

查看工作区状态

添加

git add .
git add <file>

添加文件到暂存区

提交

git commit -m "message"

提交更改到版本库

历史

git log
git log --oneline

查看提交历史

分支管理

查看分支

git branch
git branch -a

列出本地/所有分支

创建分支

git branch <name>
git checkout -b <name>

创建新分支

切换分支

git checkout <branch>
git switch <branch>

切换到指定分支

合并分支

git merge <branch>

合并指定分支

删除分支

git branch -d <branch>

删除本地分支

变基

git rebase <branch>

将当前分支变基到目标分支

远程仓库

拉取

git pull
git fetch

拉取远程更新

推送

git push
git push -u origin main

推送到远程仓库

添加远程

git remote add origin <url>

添加远程仓库

查看远程

git remote -v

查看远程仓库

撤销操作

撤销暂存

git reset <file>
git reset HEAD~1

撤销暂存区的文件

丢弃修改

git checkout -- <file>
git restore <file>

丢弃工作区修改

回退版本

git reset --hard HEAD~1

回退到上一个版本

反向提交

git revert <commit>

创建反向提交

暂存修改

git stash
git stash pop

临时保存修改

修改提交

git commit --amend

修改最后一次提交

常用技巧

配置用户信息

git config --global user.name "Your Name"
git config --global user.email "your@email.com"

查看差异

git diff
git diff --staged

标签管理

git tag v1.0
git tag -a v1.0 -m "Version 1.0"

忽略文件

echo "node_modules/" >> .gitignore
git rm --cached <file>

掌握这些Git命令，轻松进行版本控制
