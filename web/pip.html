pip 命令指南

Python包管理工具的核心命令详解

基础命令

版本信息

pip --version

查看pip版本信息

升级pip

pip install --upgrade pip

升级pip到最新版本

帮助信息

pip help
pip <command> --help

查看pip帮助信息

包管理

安装包

pip install <package>
pip install <package>==x.y.z

安装指定包或指定版本

升级包

pip install --upgrade <package>

升级已安装的包

卸载包

pip uninstall <package>

卸载指定的包

搜索包

pip search <query>

从PyPI搜索包

包信息

pip show <package>

显示包的详细信息

列出包

pip list
pip freeze

列出已安装的包

高级操作

批量安装

pip install -r requirements.txt

从文件安装多个包

导出列表

pip freeze > requirements.txt

导出已安装包列表

指定源安装

pip install -i <url> <package>

从指定镜像源安装

安装源码包

pip install <vcs+url>
pip install .

从版本控制系统或本地安装

清理缓存

pip cache purge
pip cache remove <package>

清理pip缓存

检查冲突

pip check

检查已安装包的依赖冲突

虚拟环境

创建环境

python -m venv myenv

创建虚拟环境

激活环境

source myenv/bin/activate
myenv\Scripts\activate

激活虚拟环境（Linux/macOS/Windows）

退出环境

deactivate

退出虚拟环境

常用技巧

国内镜像源设置

pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
# 清华大学镜像源

升级所有包

pip list --outdated --format=freeze | grep -v '^\-e' | cut -d = -f 1 | xargs -n1 pip install -U

安装开发版本

pip install --pre <package>
# 安装预发布版本

查看可升级的包

pip list --outdated
pip list -o

掌握这些pip命令，轻松管理Python包依赖
