Docker 命令指南

快速掌握容器化核心命令

基础命令

版本信息

docker --version

查看Docker版本

运行状态

docker info

查看Docker系统信息

帮助命令

docker --help
docker [command] --help

查看命令帮助信息

镜像管理

搜索镜像

docker search <image>

从仓库搜索镜像

拉取镜像

docker pull <image>:<tag>

下载镜像到本地

本地镜像

docker images
docker image ls

列出本地所有镜像

删除镜像

docker rmi <image>
docker image rm <image>

删除本地镜像

推送镜像

docker push <image>:<tag>

推送镜像到仓库

构建镜像

docker build -t <name> .

根据Dockerfile构建镜像

容器管理

启动容器

docker run [options] <image>

基于镜像启动新容器

查看容器

docker ps
docker ps -a

列出运行中/所有容器

停止容器

docker stop <container>

停止运行中的容器

重启容器

docker restart <container>

重启容器

删除容器

docker rm <container>
docker rm -f <container>

删除容器，-f强制删除运行中容器

进入容器

docker exec -it <container> bash

进入运行中的容器终端

高级操作

容器日志

docker logs <container>
docker logs -f <container>

查看容器日志，-f实时跟踪

容器详情

docker inspect <container>

查看容器详细信息

复制文件

docker cp <container>:<path> <host-path>

从容器复制文件到主机

端口映射

docker run -p <host-port>:<container-port>

映射容器端口到主机

数据卷

docker run -v <host-dir>:<container-dir>

挂载主机目录到容器

清理资源

docker system prune
docker system prune -a

清理未使用的资源

Docker Compose

启动服务

docker-compose up
docker-compose up -d

启动所有服务，-d后台运行

停止服务

docker-compose down

停止并移除容器

重建服务

docker-compose up --build

重建并启动服务

查看状态

docker-compose ps

查看Compose管理的容器

常用技巧

常用run参数

docker run -d --name <name> -p 80:80 -v /data:/app <image>

批量操作

docker stop $(docker ps -aq)
docker rm $(docker ps -aq)

镜像导出导入

docker save -o <file.tar> <image>
docker load -i <file.tar>

设置镜像源

# 编辑 /etc/docker/daemon.json
{"registry-mirrors":["https://mirror.example.com"]}

掌握这些Docker命令，轻松进行容器化部署
