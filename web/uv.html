UV 命令指南

Python超快包管理器和项目管理工具

项目管理

初始化项目

uv init
uv init myproject

创建新的Python项目

运行项目

uv run main.py
uv run python script.py

在项目环境中运行代码

同步依赖

uv sync
uv sync --dev

同步项目依赖到虚拟环境

锁定依赖

uv lock
uv lock --upgrade

生成或更新锁定文件

依赖树

uv tree
uv tree --depth 2

显示依赖关系树

构建项目

uv build
uv build --wheel

构建项目包

包管理

添加依赖

uv add requests
uv add "django>=4.0"

添加包到项目依赖

开发依赖

uv add --dev pytest
uv add --dev black mypy

添加开发时依赖

移除依赖

uv remove requests
uv remove --dev pytest

从项目中移除包

pip安装

uv pip install requests
uv pip install -r requirements.txt

使用uv的pip接口安装包

列出包

uv pip list
uv pip freeze

列出已安装的包

卸载包

uv pip uninstall requests
uv pip uninstall -r requirements.txt

卸载指定的包

虚拟环境

创建环境

uv venv
uv venv .venv

创建虚拟环境

激活环境

source .venv/bin/activate
# Windows: .venv\Scripts\activate

激活虚拟环境

Python版本

uv venv --python 3.11
uv venv --python python3.12

指定Python版本

查看环境

uv python list
uv python find

查看可用Python版本

工具命令

运行工具

uv tool run black .
uv tool run ruff check

临时运行工具

安装工具

uv tool install black
uv tool install ruff

全局安装工具

工具列表

uv tool list
uv tool show black

查看已安装工具

更新工具

uv tool upgrade black
uv tool upgrade --all

更新工具版本

卸载工具

uv tool uninstall black
uv tool uninstall --all

卸载全局工具

执行脚本

uvx black .
uvx --from jupyter jupyter lab

直接执行包命令

配置和缓存

清理缓存

uv cache clean
uv cache prune

清理下载缓存

缓存目录

uv cache dir

显示缓存目录路径

版本信息

uv version
uv --version

查看UV版本

帮助信息

uv --help
uv add --help

获取命令帮助

常用配置

项目配置文件 (pyproject.toml)

[project]
name = "myproject"
version = "0.1.0"
dependencies = ["requests", "click"]

开发依赖配置

[project.optional-dependencies]
dev = ["pytest", "black", "mypy"]
docs = ["mkdocs", "mkdocs-material"]

快速创建脚本

# script.py
# /// script
# dependencies = ["requests"]
# ///

设置索引源

uv pip install --index-url https://pypi.tuna.tsinghua.edu.cn/simple requests

使用UV，享受极速的Python包管理体验
