Vim 编辑器学习指南

掌握高效文本编辑的利器

关于 Vim

Vim 是一款功能强大的文本编辑器，以其高效的编辑方式和丰富的快捷键而闻名。 它的设计理念是让用户尽可能少地移动双手就能完成各种编辑任务，特别适合程序员和经常处理文本的用户。

Vim 的核心特点是其多种编辑模式，这也是初学者需要克服的第一个障碍。一旦掌握，你会发现它能极大提高你的工作效率。

Vim 模式

普通模式

默认模式，用于导航和执行命令

# 进入插入模式
i # 在光标前插入
a # 在光标后插入
o # 在当前行下方插入新行

按 ESC 从其他模式返回普通模式

插入模式

用于输入和编辑文本内容

# 常用快捷键
Ctrl + w # 删除前一个单词
Ctrl + u # 删除当前行
Ctrl + c # 返回普通模式

在该模式下，左下角会显示 -- INSERT --

命令行模式

用于执行保存、退出等命令

# 常用命令
:w # 保存文件
:q # 退出Vim
:wq # 保存并退出
:q! # 强制退出不保存

在普通模式按 : 进入该模式

导航命令

基本移动

h # 向左移动
j # 向下移动
k # 向上移动
l # 向右移动

也可以使用方向键，但推荐使用hjkl提高效率

行内移动

0 # 移动到行首
$ # 移动到行尾
w # 移动到下一个单词开头
b # 移动到上一个单词开头

在数字前加数字可指定移动次数

文件内移动

gg # 移动到文件开头
G # 移动到文件结尾
5G # 移动到第5行
Ctrl + f # 向下翻页

Ctrl + b 向上翻页，Ctrl + d 向下半页

查找

/pattern # 向下查找pattern
?pattern # 向上查找pattern
n # 下一个匹配
N # 上一个匹配

按Esc退出查找高亮

括号匹配

% # 在成对括号间跳转

适用于()、[]、{}等成对符号

标记与跳转

ma # 用a标记当前位置
`a # 跳转到a标记位置
'' # 跳回上一位置

标记可以是a-z任意字母

编辑命令

删除操作

x # 删除光标处字符
dd # 删除当前行
dw # 删除当前单词
d$ # 从光标删除到行尾

前缀数字表示操作次数，如3dd删除3行

复制粘贴

yy # 复制当前行
yw # 复制当前单词
p # 粘贴到光标后
P # 粘贴到光标前

前缀数字表示操作次数，如5yy复制5行

撤销与重做

u # 撤销上一步操作
Ctrl + r # 重做被撤销的操作

U命令撤销对整行的所有操作

替换操作

r # 替换光标处单个字符
R # 进入替换模式
:s/old/new/ # 替换当前行第一个匹配
:s/old/new/g # 替换当前行所有匹配

:%s/old/new/g 替换整个文件

缩进操作

>> # 向右缩进当前行
<< # 向左缩进当前行
=% # 自动缩进当前代码块

前缀数字表示操作行数，如3>>缩进3行

插入命令

i # 在光标前插入
I # 在行首插入
a # 在光标后插入
A # 在行尾插入
o # 在下方插入新行
O # 在上方插入新行

所有插入命令都会进入插入模式

高级功能

多窗口编辑

:split # 水平分割窗口
:vsplit # 垂直分割窗口
Ctrl + w w # 在窗口间切换
Ctrl + w q # 关闭当前窗口

可以在分割命令后指定文件名

宏录制

qx # 开始录制宏到寄存器x
... # 执行操作
q # 停止录制
@x # 执行寄存器x中的宏

数字前缀表示执行次数，如10@x执行10次

可视模式

v # 进入字符可视模式
V # 进入行可视模式
Ctrl + v # 进入块可视模式
d # 删除选中内容
y # 复制选中内容

在可视模式下可以选择文本块进行操作

外部命令

:!command # 执行外部命令
:r !date # 插入当前日期
:%!sort # 对文件内容排序

%表示整个文件内容作为输入

配置Vim

:set number # 显示行号
:set autoindent # 自动缩进
:set expandtab # 使用空格代替Tab
:set hlsearch # 高亮搜索结果

配置可写入~/.vimrc文件永久生效

插件管理

# 推荐插件管理器
Vundle
Pathogen
vim-plug
# 常用插件
NERDTree, fzf, coc.nvim

插件可以极大扩展Vim功能

实用技巧

高效操作组合

ci" # 删除引号内内容并进入插入模式
dt, # 删除到下一个逗号
gqap # 格式化当前段落
. # 重复上一个操作

必备配置

set relativenumber

set tabstop=4
set shiftwidth=4

syntax on

常用快捷键

Ctrl + u # 在插入模式删除当前行
Ctrl + y # 复制上一行相同位置内容
Ctrl + a # 增加数字
Ctrl + x # 减少数字
zz # 将当前行居中显示
