npm 命令指南

Node.js包管理工具的核心命令详解

基础命令

版本信息

npm -v
node -v

查看npm和Node.js版本信息

升级npm

npm install -g npm@latest

升级npm到最新版本

帮助信息

npm help
npm <command> -h

查看npm帮助信息

项目初始化

创建项目

npm init
npm init -y

初始化新项目，-y跳过问答直接生成

配置npm

npm config set <key> <value>
npm config get <key>

设置或获取npm配置

用户配置

npm adduser
npm whoami

登录npm账户或查看当前用户

包管理

安装包

npm install <package>
npm i <package>

安装指定包到项目

全局安装

npm install -g <package>
npm i -g <package>

全局安装指定包

卸载包

npm uninstall <package>
npm rm <package>

卸载指定的包

搜索包

npm search <keyword>

从npm仓库搜索包

包信息

npm view <package>
npm info <package>

显示包的详细信息

列出包

npm list
npm ls -g --depth 0

列出已安装的包，-g查看全局包

依赖类型

生产依赖

npm install --save <package>
npm i -S <package>

添加到dependencies（默认）

开发依赖

npm install --save-dev <package>
npm i -D <package>

添加到devDependencies

peer依赖

npm install --save-peer <package>
npm i -P <package>

添加到peerDependencies

高级操作

更新包

npm update <package>
npm upgrade <package>

更新包到最新版本

版本管理

npm version patch
npm version minor
npm version major

更新项目版本号

指定源

npm install --registry=<url>
npm config set registry <url>

使用指定镜像源安装

发布包

npm publish
npm unpublish <package>@<version>

发布或撤销发布包

清理缓存

npm cache clean --force

清理npm缓存

运行脚本

npm run <script>
npm start
npm test

运行package.json中的脚本

常用技巧

国内镜像源设置

npm config set registry https://registry.npmmirror.com
# 淘宝npm镜像源

查看可更新的包

npm outdated
# 检查过时的包

锁定依赖版本

npm shrinkwrap
# 生成npm-shrinkwrap.json锁定版本

检查包漏洞

npm audit
npm audit fix
# 检查并修复安全漏洞

掌握这些npm命令，轻松管理Node.js项目依赖
